## 실험 이론

### 커패시터의 기본 개념

- **개념**
  - 커패시터(Capacitor)는 두 개의 도체판(전극)이 유전체(절연체)를 사이에 두고 마주보는 구조로, 전기 에너지를 **전기장(Electric Field)**의 형태로 저장하는 수동 소자이다.
  - 커패시턴스(Capacitance) C는 커패시터가 전하를 저장할 수 있는 능력을 나타내며, 단위는 패럿(F, Farad)이다.
  - 평행판 커패시터의 커패시턴스는 다음과 같이 결정된다:
    - C = 8.85 × 10⁻¹² × εᵣ × (A / d)  [F] ··· (15.1)
    - εᵣ: 유전체의 비유전율 (공기 = 1)
    - A: 전극판의 면적 [m²]
    - d: 전극판 사이의 거리 [m]
  - 커패시터에 저장된 전하량 Q와 전압 V의 관계:
    - Q = CV  [C, 쿨롱] ··· (15.2)

- **적용 방법**
  - 본 실험에서는 전해 커패시터(100 μF, 220 μF)를 사용하며, **극성이 있으므로** 마이너스 단자를 접지 방향으로 연결해야 한다.
  - 커패시턴스 값이 클수록 더 많은 전하를 저장할 수 있으며, 충전/방전 시간이 길어진다.

### 커패시터의 직렬 및 병렬 연결

- **개념**
  - **직렬 연결**: 전극판 간 거리가 증가하는 효과 → 총 커패시턴스가 **감소**한다.
    - 1/C_T = 1/C₁ + 1/C₂ + 1/C₃ + ··· ··· (15.3)
    - (병렬 저항의 합성 공식과 동일한 형태)
  - **병렬 연결**: 전극판의 총 면적이 증가하는 효과 → 총 커패시턴스가 **증가**한다.
    - C_T = C₁ + C₂ + C₃ + ··· ··· (15.4)
    - (직렬 저항의 합성 공식과 동일한 형태)

- **적용 방법**
  - Ch 16 Part 1에서 C₁ = 100 μF과 C₂ = 220 μF이 **병렬 연결**되어 있으므로:
    - C_T = C₁ + C₂ = 100 + 220 = 320 μF (명판값 기준)
  - 시정수 계산 시 합성 커패시턴스 C_T를 사용한다.

### 커패시터에 저장되는 에너지

- **개념**
  - 커패시터에 저장되는 에너지는 다음과 같다:
    - W = (1/2) × C × V²  [J, 줄] ··· (15.5)
  - 저항이 열로 에너지를 소비하는 것과 달리, 커패시터는 에너지를 **저장**하고 나중에 방출할 수 있다.

- **적용 방법**
  - Ch 15 Part 1에서 정상상태 시 V_C = E = 10 V이므로:
    - W = (1/2) × 100 × 10⁻⁶ × 10² = 5 × 10⁻³ J = 5 mJ
  - 전원을 분리한 후에도 커패시터에 전압이 남아 있으므로, 실험 전후 반드시 방전시켜야 한다.

### DC 정상상태 해석 (Steady-State Analysis)

- **개념**
  - DC 회로에서 커패시터는 충전이 완료되면(5τ 이상 경과) 더 이상 전류가 흐르지 않는다.
  - 따라서 DC 정상상태에서 **커패시터는 개방 회로(Open Circuit)**로 취급한다.
  - 이는 커패시터를 통한 전류가 i_C = C × (dv_C/dt) = 0 이기 때문이다 (정상상태에서 dv_C/dt = 0).
  - 인덕터의 경우 DC 정상상태에서 **단락 회로(Short Circuit)**로 취급한다 (참고).

- **적용 방법**
  - **Part 1 (직렬 R-C)**: 커패시터를 개방 → 회로에 전류가 흐르지 않음 → I = 0, V_R = 0 V, V_C = E = 10 V
  - **Part 2 (병렬 R-C)**: 커패시터 분기를 개방 → R₁과 R₂의 직렬 회로로 단순화 → 전압 분배 법칙 적용
  - **Part 3 (직병렬 R-C)**: 두 커패시터를 모두 개방 → 모든 전류 경로가 차단됨 → 각 소자의 전압·전류 분포 분석

### 시정수 (Time Constant, τ)

- **개념**
  - R-C 회로에서 시정수 τ는 과도 응답의 속도를 결정하는 파라미터이다:
    - τ = R × C  [s, 초] ··· (15.6)
  - 1τ 경과 시: 최종값의 **63.2%**에 도달 (충전 시)
  - 2τ 경과 시: 86.5%
  - 3τ 경과 시: 95.0%
  - 4τ 경과 시: 98.2%
  - 5τ 경과 시: 99.3% → 실질적으로 **정상상태에 도달**했다고 간주

- **적용 방법**
  - Ch 15 Part 4에서 시정수를 실험적으로 측정하여 커패시터의 실제 커패시턴스를 역산한다:
    - C_measured = τ_measured / R_measured ··· (15.7)
  - τ 측정 방법: 전원 인가 후 v_C가 0.632 × E (= 6.32 V, E = 10 V 기준)에 도달하는 시간을 스톱워치로 측정

### 과도 응답 (Transient Response)

- **개념**
  - R-C 회로에 DC 전원을 인가하면, 커패시터 전압은 즉시 최종값에 도달하지 않고 **지수 함수적으로 변화**한다. 이 구간을 과도 응답(Transient Response)이라 한다.
  - 충전 시 (전원 인가):
    - 커패시터 전압: v_C = E × (1 − e^(−t/RC))  [V]
    - 저항 전압: v_R = E × e^(−t/RC)  [V]
    - 회로 전류: i_C = (E/R) × e^(−t/RC)  [mA]
  - 핵심 관계: v_R = E − v_C (KVL에 의해)
  - 충전 과정의 지속 시간은 약 5τ이며, 이후 정상상태에 도달한다.

- **적용 방법**
  - Ch 16 Part 1에서 R = 100 kΩ, C₁ ∥ C₂ = 100 μF + 220 μF = 320 μF (명판값):
    - τ = 100 × 10³ × 320 × 10⁻⁶ = 32 s
    - 5τ = 160 s
  - 지정된 시간 간격(0, 10, 20, 30, ... 초)에서 v_C 값을 수식으로 계산하고, 실험 측정값과 비교한다.
  - t = τ에서 v_C = 0.632 × 12 = 7.58 V가 되는지 확인한다.

### 옴의 법칙과 전압 분배 법칙

- **개념**
  - 옴의 법칙: V = I × R
  - 직렬 회로의 전압 분배: V_x = (R_x / R_total) × E
  - 직렬 회로의 전류: I = E / R_total

- **적용 방법**
  - 정상상태에서 커패시터를 개방 회로로 치환한 후 남은 저항 네트워크에 적용한다.
  - Ch 15 Part 2: R₁과 R₂가 직렬 → V₁ = R₁/(R₁+R₂) × E, V₂ = R₂/(R₁+R₂) × E
  - 전류 계산: I = E / (R₁ + R₂)

### KVL (키르히호프 전압 법칙)

- **개념**
  - 임의의 폐회로(Closed Loop)를 따라 한 바퀴 돌 때, 전압 상승의 합 = 전압 강하의 합
  - 즉, 폐회로 내 모든 전압의 대수적 합 = 0

- **적용 방법**
  - 모든 예상값 계산 후 KVL 검증에 사용:
    - 직렬 R-C: V_R + V_C = E
    - 병렬 R-C: V₁ + V₂ = E (R₁, R₂ 직렬 루프)
  - 계산 결과의 정확성을 확인하는 필수 검증 도구

### KCL (키르히호프 전류 법칙)

- **개념**
  - 임의의 노드(Node)에서 유입되는 전류의 합 = 유출되는 전류의 합
  - 즉, 한 노드에서 모든 전류의 대수적 합 = 0

- **적용 방법**
  - Ch 15 Part 2 (Fig 15.2): 분기점에서 I₁ = I₂ + I₃
    - 정상상태에서 I₃ = 0 (커패시터 개방) → I₁ = I₂
  - Ch 15 Part 3 (Fig 15.3): 각 노드에서 전류 보존 확인

    - E = V₁ + V₂
    - 10 = 0 + V₂
    - V₂ = **10 V**

- KVL 검증:
  - V₁ + V₂ = 0 + 10 = 10 V = E ✅

#### Table 15.2 (Part 2: Parallel R-C dc Network)

- 회로 소자값 정리
  - R₁ = 1.2 kΩ, R₂ = 3.3 kΩ, C = 100 μF, E = 10 V

- [Topology] Fig 15.2
  - node A: R₁ 출력, R₂ 상단, C 상단 연결 (분기점)
  - E⁺ —[R₁ = 1.2 kΩ]→ node A —[R₂ = 3.3 kΩ]→ GND (I₂ 경로)
  - node A —[C = 100 μF]→ GND (I₃ 경로)

  - 커패시터를 개방 회로로 치환 → I₃ = 0 mA

    - R_total = R₁ + R₂ = 1.2 + 3.3 = 4.5 kΩ
  - 전류 계산:
    - I₁ = I₂ = E / R_total = 10 / 4.5 kΩ = **2.22 mA**
  - 전압 분배 법칙:
    - V₁ = E × R₁ / R_total = 10 × 1.2 / 4.5 = **2.67 V**
    - V₂ = E × R₂ / R_total = 10 × 3.3 / 4.5 = **7.33 V**
  - C는 R₂와 병렬이므로:
    - V₃ = V₂ = **7.33 V**

- KVL 검증 (E → R₁ → R₂ → GND 루프):
  - V₁ + V₂ = 2.67 + 7.33 = 10.00 V = E ✅

- KCL 검증 (node A):
  - I₁ = I₂ + I₃ = 2.22 + 0 = 2.22 mA ✅

  - 양쪽 커패시터를 모두 개방 회로로 치환한다.

    - 경로 1: node A → C₁ → node B → R₂ → GND → C₁ 개방으로 차단
    - 경로 2: node A → C₂ → GND → C₂ 개방으로 차단
  - 따라서 모든 전류가 0:
    - I₁ = I₂ = I₃ = I₄ = **0 mA**

  - 전압 계산:
    - V₁ = I₁ × R₁ = 0 × 1.2 kΩ = **0 V**
    - V₂ = I₂ × R₂ = 0 × 3.3 kΩ = **0 V**
    - node A의 전위: V_A = E − V₁ = 10 − 0 = 10 V
    - node B의 전위: V_B = V₂ = 0 V (R₂에 전류가 흐르지 않으므로 node B = GND)
    - V₃ = V_A − V_B = 10 − 0 = **10 V** (C₁ 양단 전압)
    - V₄ = V_A − GND = 10 − 0 = **10 V** (C₂ 양단 전압)

- KVL 검증:
  - 루프 1 (E → R₁ → C₂ → GND): V₁ + V₄ = 0 + 10 = 10 V = E ✅
  - 루프 2 (E → R₁ → C₁ → R₂ → GND): V₁ + V₃ + V₂ = 0 + 10 + 0 = 10 V = E ✅
  - 루프 3 (C₂ ∥ C₁–R₂): V₄ = V₃ + V₂ → 10 = 10 + 0 ✅

- KCL 검증 (node A):
  - I₁ = I₃ + I₄ → 0 = 0 + 0 ✅
- KCL 검증 (node B):
  - I₃ = I₂ → 0 = 0 ✅
